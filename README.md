# Sociability-Cages
A repo for stereolithography files for constructing mouse holders for the 3-Chamber Social Interaction test developed by Crawley et al.

Print in 100% infill, and seal in polyurethane to prevent odor impregnation in between plastic layers.

This design relies on 3mm acrilyc rods of 15cm in length.
